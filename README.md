# Sky: Children of the Light — KTX v1 + BC7 Texture Toolchain (Decoder & Encoder) 🕯️

Универсальный набор инструментов на Python для точного анализа, декодирования, кодирования и конвертации кастомных `.ktx` текстур из ПК-версии игры *Sky: Children of the Light*.

> **Статус:** Полный рабочий цикл моддинга для PC (BC7) — Чтение и Запись 🚀 

После 3 месяцев хардкорного реверс-инжиниринга, разбора байт-кода и спецификаций Khronos/Microsoft, алгоритм блочного сжатия BC7 был полностью побежден. Инструменты реализованы на чистом Python, работают без тяжелых внешних библиотек и проверены прямо в игре!

## 🌟 Особенности алгоритмов

### Декодер (`ktx_bc7_decoder.py`):
- **100% точный побитовый декод** для BC7 модификаций (Mode 4, 5, 6) строго по официальной спецификации Microsoft/Khronos BPTC.
- **Приближенное качественное чтение** для мульти-сабсет модов (Mode 0, 1, 2, 3, 7) через усреднение эндпоинтов блоков. Больше никакого "шума" и диагональных артефактов на ПК-текстурах!

### Энкодер (`ktx_bc7_encoder.py`):
- **Быстрое и корректное кодирование** любых изображений в формат BC7 (всегда используется Mode 6: single-subset, эндпоинты 7-бит + P-bit).
- **Автоматический репакинг**: скрипт берет заголовок оригинального игрового `.ktx`, обновляет в нем размеры и аккуратно упаковывает новые BC7-данные обратно. Движок игры принимает такие текстуры без ошибок!

## 🚀 Инструкция по использованию

### 1. Как распаковать текстуру из игры в обычную картинку:
```bash
python ktx_bc7_decoder.py имя_текстуры.ktx готовая_картинка.png
```

### 2. Как запаковать измененную картинку обратно в игру:
Возьмите вашу новую картинку (PNG или JPG), оригинальный KTX-файл игры (оттуда скрипт заберет структуру заголовка) и укажите имя нового файла:
```bash
python ktx_bc7_encoder.py новая_картинка.png оригинал.ktx итоговый_файл.ktx
```
*После этого просто переименуйте итоговый файл в оригинальное имя и закиньте в папку игры с заменой!*
