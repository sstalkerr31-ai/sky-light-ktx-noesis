# Sky: Children of the Light — KTX v1 + BC7 Texture Toolchain & GUI Mod Kit 🕯🎨

Универсальный набор инструментов на Python (CLI + GUI) для точного анализа, декодирования, кодирования и замены `.ktx` текстур в ПК-версии *Sky: Children of the Light*. Алгоритмы BC7 полностью реверс-инжинирингованы, обеспечивая точное чтение (Mode 4-6) и корректную сборку (Mode 6) без тяжелых библиотек.

---

## 🖥 Новинка: Графический мод-кит (`ktx_tool_gui.py`)

Удобный интерфейс для безопасного моддинга:
- **Decode/Encode:** Декодирование в PNG, обратное кодирование в BC7 и внедрение в игру.
- **Safe Modding:** Автоматическое создание бэкапов (`.ktx.bak`).
- **Threading:** Тяжелые операции вынесены из основного потока, GUI не зависает.

---

## 🌟 Особенности алгоритмов (CLI)

- **Декодер (`ktx_bc7_decoder.py`):** Побитовый декод BC7, обработка мульти-сабсетов, качественное отображение текстур.
- **Энкодер (`ktx_bc7_encoder.py`):** Кодирование в BC7 (Mode 6) с автоматическим репакингом заголовков для полной совместимости с движком.

---

## 🛠 Требования и использование

Требуется: `pip install Pillow`

1.  **GUI:** `python ktx_tool_gui.py` (указать путь к папке текстур).
2.  **CLI Декодинг:** `python ktx_bc7_decoder.py <ktx> <png>`
3.  **CLI Энкодинг:** `python ktx_bc7_encoder.py <png> <old_ktx> <new_ktx>`
